## 🚀 Быстрый старт

### Локальный запуск

```bash
# Клонировать репозиторий
git clone https://github.com/Piristo/ux-dashboard.git
cd ux-dashboard

# Запустить простой HTTP сервер
python -m http.server 8000
# или
npx serve

# Открыть в браузере
http://localhost:8000
```

### Требования
- Современный браузер (Chrome, Firefox, Safari, Edge)
- HTTP сервер (для Service Worker)

## 🎨 3D Анимация

Проект включает красивую 3D фоновую анимацию:
- 50 плавающих частиц с перспективой
- Волновые эффекты
- Взаимодействие с курсором мыши
- Динамические соединения между частицами
- 60 FPS производительность

### Настройка анимации

Откройте `background-animation.js` и измените параметры:

```javascript
const config = {
    particleCount: 50,              // Количество частиц
    particleSpeed: { min: 0.3, max: 1.5 },
    waveCount: 3,                   // Количество волн
    mouseRadius: 150,               // Радиус взаимодействия
};
```

## 🏷️ Теги

Теги помогают организовать заметки:
- Добавляйте до 10 тегов к каждой заметке
- Вводите теги и нажимайте Enter
- Теги отображаются как красивые badges

**Примеры тегов:** `работа`, `важно`, `срочно`, `личное`, `покупки`

## 📱 Установка как PWA

### На десктопе:
1. Откройте приложение в Chrome
2. Кликните на иконку установки в адресной строке
3. Нажмите "Установить"

### На мобильном:
1. Откройте в Safari/Chrome
2. Нажмите "Поделиться" → "На экран Домой"
3. Назовите ярлык и добавьте
