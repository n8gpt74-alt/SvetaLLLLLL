# 📅 Sveta Calendar - Информация о календаре

## ✅ Подтверждение: Календарь начинается с ПОНЕДЕЛЬНИКА

Ваш календарь **уже правильно настроен** и начинается с понедельника!

### Как это работает?

#### 1. **HTML структура (index.html)**
```html
<div class="weekdays">
    <span>Пн</span>  <!-- Понедельник - первый день -->
    <span>Вт</span>
    <span>Ср</span>
    <span>Чт</span>
    <span>Пт</span>
    <span>Сб</span>
    <span>Вс</span>  <!-- Воскресенье - последний день -->
</div>
```

#### 2. **JavaScript логика (app.js, строки 311-315)**
```javascript
// Рассчитываем смещение для начала недели с понедельника
// getDay() возвращает: 0=Вс, 1=Пн, 2=Вт, 3=Ср, 4=Чт, 5=Пт, 6=Сб
// Вычитаем 1, чтобы Пн=0, Вт=1, ..., Вс=6
let startDay = firstDay.getDay() - 1;
if (startDay < 0) startDay = 6; // Если воскресенье (0-1=-1), то делаем 6
```

### Объяснение логики

В JavaScript метод `Date.getDay()` возвращает:
- **0** = Воскресенье
- **1** = Понедельник
- **2** = Вторник
- **3** = Среда
- **4** = Четверг
- **5** = Пятница
- **6** = Суббота

Чтобы календарь начинался с понедельника, мы:
1. Вычитаем 1 из результата `getDay()`
2. Если получается -1 (воскресенье), мещаем его в конец (6)

### Примеры

| Первый день месяца | getDay() | Расчёт | Смещение | Результат |
|-------------------|----------|---------|----------|-----------|
| Понедельник       | 1        | 1-1=0   | 0        | Начинается с первой колонки (Пн) |
| Вторник           | 2        | 2-1=1   | 1        | Начинается со второй колонки (Вт) |
| Среда             | 3        | 3-1=2   | 2        | Начинается с третьей колонки (Ср) |
| Воскресенье       | 0        | 0-1=-1→6| 6        | Начинается с седьмой колонки (Вс) |
