## ⚙️ **Настройка анимации**

Откройте `background-animation.js` и измените конфигурацию:

```javascript
const config = {
    particleCount: 50,              // Количество частиц
    particleSize: { min: 2, max: 6 },
    particleSpeed: { min: 0.3, max: 1.5 },
    particleColors: ['#4f8cff', '#a855f7', '#f97316', '#3b7aed'],
    waveCount: 3,                   // Количество волн
    mouseRadius: 150,               // Радиус взаимодействия
    connectionDistance: 120         // Расстояние соединений
};
```

### Рекомендации:
- **Больше частиц?** - Увеличьте до 80 (если не тормозит)
- **Медленнее?** - Уменьшите particleSpeed.max до 1.0
- **Другие цвета?** - Измените particleColors
- **Тормозит?** - Уменьшите particleCount до 30
