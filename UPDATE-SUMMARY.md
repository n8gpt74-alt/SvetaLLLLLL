## 🎯 **Как протестировать**

### Теги:
1. Обновите страницу (F5)
2. Выберите дату
3. Создайте новую заметку
4. В поле "Теги" введите тег и нажмите Enter
5. Добавьте ещё несколько тегов
6. Сохраните заметку
7. Проверьте, что теги отображаются внизу карточки

### Редактирование тегов:
1. Откройте существующую заметку для редактирования
2. Добавьте или удалите теги
3. Сохраните изменения
4. Убедитесь, что зменения сохранились

### Теги лучше использовать для:
- **Категоризации** (работа, личное, дом)
- **Приоритетов** (срочно, важно, мелочи)
- **Контекста** (звонки, email, встречи)
- **Проектов** (проект-а, проект-б)
- **Людей** (иван, мария, команда)

