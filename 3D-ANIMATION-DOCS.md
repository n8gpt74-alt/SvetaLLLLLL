# 🎨 3D Фоновая Анимация

## ✨ **Что добавлено**

Крутая 3D анимация фона с плавающими частицами для Sveta Calendar!

### Особенности:
- ✅ **50 плавающих 3D частиц** с эффектом перспективы
- ✅ **Волновой эффект** на фоне (3 волны с разной скоростью)
- ✅ **Интерактивность** - частицы реагируют на курсор мыши
- ✅ **Соединения** - линии между близкими частицами
- ✅ **Эффект свечения** вокруг каждой частицы
- ✅ **Glassmorphism** - полупрозрачные элементы UI с размытием
- ✅ **60 FPS** - плавная анимация на Canvas
- ✅ **Адаптивность** - подстраивается под размер экрана

---

## 🎭 **Технические детали**

### Canvas-анимация:
- **Canvas 2D API** для отрисовки
- **RequestAnimationFrame** для плавной анимации
- **3D перспектива** через ось Z (глубина 0-1000)
- **Gradient effects** для свечения частиц

### Частицы:
```javascript
{
    x, y, z              // Позиция в 3D пространстве
    speedX, speedY, speedZ  // Скорость движения
    size                 // Размер частицы (2-6px)
    color                // Один из 4 цветов
    opacity              // Прозрачность (0.3-0.7)
}
```

### Волны:
- 3 синусоидальные волны с разной частотой
- Различная амплитуда и скорость
- Цвета из палитры приложения

### Взаимодействие с мышью:
- Радиус отталкивания: **150px**
- Частицы уклоняются от курсора
- Плавное перемещение с физикой

### Соединения:
- Линии между частицами < **120px**
- Opacity зависит от расстояния
- Синий цвет (#4f8cff)

## ⚙️ **Конфигурация**

Настройки анимации в `background-animation.js`:

```javascript
const config = {
    particleCount: 50,              // Количество частиц
    particleSize: { min: 2, max: 6 },  // Размер частиц
    particleSpeed: { min: 0.3, max: 1.5 }, // Скорость
    particleColors: [                // Цвета
        '#4f8cff',  // Синий
        '#a855f7',  // Фиолетовый
        '#f97316',  // Оранжевый
        '#3b7aed'   // Темно-синий
    ],
    waveCount: 3,                   // Количество волн
    mouseRadius: 150,               // Радиус взаимодействия с мышью
    connectionDistance: 120         // Расстояние для соединений
};
```

### Как изменить:

1. **Больше частиц** - увеличьте `particleCount` (но учтите производительность)
2. **Медленнее** - уменьшите `particleSpeed.max`
3. **Другие цвета** - измените `particleColors`
4. **Меньше волн** - уменьшите `waveCount`

## 🎨 **Визуальные эффекты**

### 1. **3D Перспектива**
Частицы движутся по оси Z:
- Близкие частицы - **большие и яркие**
- Дальние частицы - **маленькие и тусклые**
- Создаёт иллюзию глубины

### 2. **Радиальное свечение**
Каждая частица имеет:
- Яркое ядро
- Gradient glow вокруг (3x размер)
- Fade out к краям

### 3. **Динамические соединения**
Линии между частицами:
- Появляются/исчезают плавно
- Прозрачность зависит от расстояния
- Создают "сеть"

### 4. **Волновой фон**
Синусоидальные волны:
- 3 слоя с разной скоростью
- Создают глубину
- Едва заметные (opacity: 0.03-0.06)

## 💡 **Советы**

### Если анимация тормозит:
1. Уменьшите `particleCount` до 30
2. Уменьшите `connectionDistance` до 80
3. Отключите волны (`waveCount: 0`)

### Если хотите ярче:
1. Увеличьте `opacity` частиц (в коде Particle class)
2. Добавьте больше цветов в `particleColors`
3. Увеличьте `waveCount` и их opacity
